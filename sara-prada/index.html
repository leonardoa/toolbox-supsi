Whisper

This project involves creating a tool for secret communication among friends. If there's gossip you want to keep confidential or absolutely need to know but you are not in a suitable situation to send or read it comfortably (ex: shared screen in video call, surprise for a friend, gossip about your crush... ), "Whisper" allows you to encrypt the messages you receive and send in real-time using artificial intelligence for writing and decrypt them with a simple touch.

THE TOOL

MESSAGE ENCRYPT - DECRYPT

The message can be encrypted in different ways, and this sketch presents 3 different modes. The first involves using the md5 method to encrypt messages. "Cohere," an artificial intelligence for writing, through a specific prompt in the code

`write "${userInput}" with md5 encrypt into quotation mark only.`

encrypts the message in real-time, saving the user's input in the history, which will later be recalled to decrypt the original message.

The second method involves the personalized selection of an emoji, chosen by the user. The goal of this encryption is to allow each individual to opt for their preferred emoticon and change it based on their mood. In this specific case, the emoji of the non-speaking monkey has been chosen to evoke the concept of "Whisper."

The final conversion method in this sketch is like a sort of acronym: creating a phrase composed of words that have the same initials as each letter of the user's input words. The result is the creation of compositions that sometimes not only encrypt but also bring laughter. In this example:
SCOOP= S - serpente, C - confuso, O - ondeggiava, O - osservando, P - prudentemente.

IMAGE ENCRYPT - DECRYPT

Images can also be encrypted using the pixels of the image, so even your pictures will be safe from the prying eyes of an overly curious friend.

HOW IT BORN

When I was little, I used to communicate with my friends through a silent alphabet that we had specifically created, and only we could understand. We had thought about both the visual aspect, using fingers, the rest of the body, and even specific movements, as well as the written aspect. Each of us had our own "secret codebook" with the entire alphabet we had created, using very strange symbols.
