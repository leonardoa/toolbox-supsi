ShopSaver

ShopSaver is a series of Chrome extensions conceived for shopping websites. The purpose of this creative coding project is to prevent people from shopping online and wasting money on things they don’t need, especially during sales when individuals tend to go overboard with discounts. Each extension has different interactions with the goal of not allowing users to select a product or complete an order. Some interactions start automatically, while others require an 'input', such as mouse hover, for example.


  • Link Github

How to use them

To try the extensions, it is necessary to download all the 'ShopSaver' folders and then upload each one of them in Chrome. Once they appear in your extensions bar, you need to select the icon of the extension you want to use on the top-right corner of the search engine page and then click on the 'ShopSaver' button to activate it.

Tool presentation

EXTENSION ICONS

I chose the snake as the icon because of its shape, which resembles the letter 'S,' representing both 'Shop' and 'Saver.' Additionally, the hissing sound made by the snake aligns with the 'S' sound. Each icon features a different color to distinguish between versions in the series.
