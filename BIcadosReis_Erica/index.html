Smell it camera

This coding project is centered around developing an innovative tool. In my case, the implemented tool is an application designed to assist individuals, including myself, who experience anosmia, allowing them to "sense" smells once again. The application operates by utilizing the phone's camera, which is connected to software capable of detecting objects in the captured image (utilizing Google Vision). From this detection, a description of the potential smell associated with that object is generated through Cohere.

THE PROCESS

The process of developing this tool was lengthy. First, I began with a brainstorming session to understand how it could function. I started by researching and analyzing other case studies. During these investigations, I proceeded to implement the code in steps. Initially, I focused on object detection in the camera using TensorFlow. After that, I conducted the same experiment with Google Vision and realized it was the better choice. Once I completed this portion, I searched for software that would allow me to generate descriptions using AI and settled on Cohere. Throughout these stages, I encountered various issues with API keys that would stop functioning. I added features I desired, such as archiving both images and descriptions, and the inclusion of date and time for each capture.
