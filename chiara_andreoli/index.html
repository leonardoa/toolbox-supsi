Toolbox
WS – SUPSI Bachelor of Arts SUPSI in Visual communication.

Toolbox coding week workshop held by Leonardo Angelucci and Desirée Veschetti with the third-year students at the Bachelor of Arts in Visual communication at SUPSI. The project focuses on experimenting with contemporary technologies such as machine learning through targeted tools (ml5, posNet, Javascript, p5js, Arduino, ChatGPT, Elevenlabs).
The primary goal of this workshop was to guide students through the development of various tools using coding techniques, with a focus on creating applications that operate within web browsers. This approach allowed for the creation of both utilitarian and experimental tools, emphasizing accessibility and practicality.
